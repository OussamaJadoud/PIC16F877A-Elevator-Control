# Proteus Simulation - Elevator Project

## Files
- `elevator.pdsprj` : Proteus project file
- `elevator.hex` : compiled program (from C or ASM)
- `schematic.png` : schematic diagram

## Instructions
1. Open `elevator.pdsprj` with Proteus
2. Make sure the `.hex` file is loaded
3. Run the simulation
4. Press the floor buttons to test the elevator control
